🧾 メモとまとめ

🔹 Gitとは？

Gitはバージョン管理システムで、ファイルの変更履歴を記録し、チームでの開発を効率化します。

🔹 よく使うコマンド

  • git init：新しいリポジトリを作成
  • git add：ファイルをステージに追加
  • git commit：ステージされた変更を記録
  • git push：リモートリポジトリに送信

🧪 演習記録

演習1： Gitの初期化とコミット
git init
git add index.html
git commit -m "初回コミット"
演習2： GitHubへのpush
git remote add origin https://github.com/username/repo.git
git push -u origin main
